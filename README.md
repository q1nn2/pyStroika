# Стройка — мини-игра для обучения Python детям

Веб-приложение: строительная тематика, 7 уровней от вывода до функций и списков, редактор кода, проверка решений, рейтинг и прогресс.

## Играть онлайн

**[https://q1nn2.github.io/pyStroika/](https://q1nn2.github.io/pyStroika/)**

## Локальный запуск

1. Запусти сервер: дважды щёлкни по `start-server.bat` или выполни в терминале: `python -m http.server 8000`
2. Открой в браузере: http://localhost:8000

## Структура проекта

- `index.html`, `game.html`, `progress.html` — страницы игры
- `css/main.css` — стили
- `js/` — логика (редактор, выполнение кода, проверка, прогресс)
- `data/levels.json` — задания и тесты
