# Стройка — мини-игра для обучения Python детям

Веб-приложение: строительная тематика, 7 уровней от вывода до функций и списков, редактор кода, проверка решений, рейтинг и прогресс.

## Играть онлайн (GitHub Pages)

После публикации: **https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПО/**

## Как запустить локально

1. Запусти сервер: дважды щёлкни по `start-server.bat` или в терминале: `python -m http.server 8000`
2. Открой в браузере: **http://localhost:8000**

## Структура

- `index.html`, `game.html`, `progress.html` — страницы
- `css/main.css` — стили
- `js/` — логика (editor, runner, checker, progress и др.)
- `data/levels.json` — задания и тесты
